# Frases Úteis em Holandês

Site estático com mais de 30 temas de frases em holandês para iniciantes, com áudio nativo (Web Speech API).

## Características

- 32 temas de frases organizados em duas abas (Frases Úteis 1 e Frases Úteis 2)
- Áudio em holandês nativo via navegador (Web Speech API, voz nl-NL)
- Selector de voz e controle de velocidade (0.5x a 1.2x)
- Perguntas e respostas com cores diferentes para facilitar o estudo
- Sumário clicável para navegação rápida
- Design responsivo com paleta de cores da bandeira holandesa

## Como publicar no Vercel

1. Crie um novo repositório no GitHub.
2. Faça upload do arquivo `index.html` (e opcionalmente este `README.md`).
3. No Vercel, clique em "Add New Project" e importe o repositório.
4. Não é necessário configurar build, é HTML puro.
5. Pronto. Seu site fica online em poucos segundos.

## Tecnologias

- HTML5
- CSS3 (sem frameworks)
- JavaScript vanilla
- Web Speech API (síntese de voz nativa do navegador)
